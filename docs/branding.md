# KokinOS Branding

KokinOS uses a coqui frog mascot and a tropical-but-polished color palette:

- Frog green: `#39d98a`
- Deep charcoal: `#101820`
- Clean off-white: `#f5f7ef`
- Accent blue: `#2aa8ff`
- Accent gold: `#f5c542`

The v1 image includes an original SVG mascot/logo, KDE theme metadata, an icon theme overlay, a Plymouth boot splash, an SDDM theme, and a welcome page.

The first public beta should replace the current starter SVGs with finalized mascot artwork before announcement.
